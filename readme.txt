== Description ==

Page popup plugin for popup, provide a simple and fast popup for wordpress website.

It will take just 2 minute to setup popup in you wordpress website.

= Available Popup Animation Effects in this Version =
Fade in, Slide in (right), Slide in (bottom), Newspaper, Fall, Side Fall, Sticky Up, 3D Flip (horizontal), 3D Flip (vertical), 3D Sign, Super Scaled, Just Me, 3D Slit, 3D Rotate Bottom, 3D Rotate In Left, Blur, Let me in, Make way!, Slip from top

== Installation ==

For the plugin installation follow the below steps:

1. Unzip and Upload the directory ‘Popup’ to the ‘/wp-content/plugins/’ directory.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings -> Popup Plugin Settings menu and add popup as needed

== Backend ==

For add popup Go to  Settings -> Popup Plugin Settings page. On this page you need to add following data for Popup:

1. Popup Title
2. Popup Title Colour
3. Popup Description
4. Popup Description Colour
5. Popup Button Text
6. Popup Button Link
7. Popup Button Colour
8. Popup Button Text Colour
9. Popup Background Colour
10. Popup Background Image
11. Popup Timeout (Enter time in millisecond “1 Second = 1000 Millisecond”)
12. Popup Width
13. Popup Height
14. Display Popup one time only?
15. Popup Effect
16. Overlay Colour
17. Overlay Opacity

Now, Go to particular page on which page you want to display Popup and select checkbox “Allow Popup” for display Popup on that page.

== Frontend ==

Popup will display on particular frontend page after particular timeout which given in backend settings page.

== Frequently Asked Questions ==

= How to install this plugin? =
Check out Installation section for more details about plugin installation.

= How to change timeout of Autoload popup? =
Go to Settings -> Popup Plugin Settings page and change "Popup Timeout" there. Please note that add timeout in Millisecond.

= How to set background colour for popup? =
For set background colour first remove background image and add background colour in plugin settings page.

= How to display popup on everytime when user load page in frontend? =
Disable "Display Popup one time only?" setting in plugin settings page for display popup on everytime when user load page in frontend.
